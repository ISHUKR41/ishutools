### 8 Intelligent Split Modes

```
┌──────────────────┬────────────────────────────────────────────────────────┐
│ Mode             │ Description                                             │
├──────────────────┼────────────────────────────────────────────────────────┤
│ All Pages        │ Burst every page into its own PDF — packed in ZIP       │
│ Page Range       │ Extract specific pages ("1-5, 8, 12-end, odd, even")    │
│ Range Groups     │ Multiple ranges → separate PDFs (EXCLUSIVE to IshuTools)│
│ Every N Pages    │ Split into equal chunks of N pages                      │
│ By Bookmarks     │ Each chapter/bookmark → its own named PDF file          │
│ Blank Separator  │ Auto-detect blank divider pages (pixel-level analysis)  │
│ By File Size     │ Split so each part ≤ chosen MB — perfect for email      │
│ Odd / Even       │ Separate odd-numbered and even-numbered pages            │
└──────────────────┴────────────────────────────────────────────────────────┘
```

### Lossless Quality — The Core Principle

```python
# IshuTools Split PDF — Ishu Kumar's lossless guarantee
import pikepdf

with pikepdf.open(input_path) as src:
    dst = pikepdf.Pdf.new()
    for page_idx in selected_pages:
        dst.pages.append(src.pages[page_idx])
    # recompress_flate=False → byte-perfect copy, zero re-encoding
    dst.save(output_path, recompress_flate=False)
```

> **This is what separates IshuTools from every competitor.**
> `recompress_flate=False` means images, fonts, and embedded objects are **byte-copied** without any re-encoding whatsoever. The output is **bit-for-bit identical** in quality to the input — always, without exception.

## 📡 API Reference

### Split PDF — Complete API

#### `POST /api/split-pdf` — Main split operation

**Request (multipart/form-data):**
| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `file` | File | ✅ | PDF file to split |
| `mode` | string | ✅ | `all` \| `range` \| `range_groups` \| `every_n` \| `bookmarks` \| `blank_pages` \| `size_limit` \| `odd_even` |
| `range` | string | mode=range | `"1-5, 8, 12-end, odd, even, first 10"` |
| `range_groups` | string | mode=range_groups | `"1-5\n6-10\n11-20"` (newline-separated) |
| `n` | integer | mode=every_n | Pages per chunk |
| `size_mb` | float | mode=size_limit | Max MB per output file |
| `password` | string | | For encrypted PDFs |
| `naming_pattern` | string | | Default: `page_{n:04d}`. Supports `{n}`, `{date}` |
| `remove_blanks` | string | | `"true"` to remove blank pages from output |
| `blank_thresh` | integer | | 80–99, default 94 (mode=blank_pages) |

**Response Headers:**
```
X-Total-Pages      : total pages in source PDF
X-Method-Used      : pikepdf | ghostscript | fallback
X-Output-Size      : ZIP file size in bytes
X-Quality-Score    : 0-100 quality score
X-Quality-Grade    : A+ | A | B | C | D
X-Processing-Ms    : processing time in milliseconds
X-File-Names       : pipe-separated output filenames
X-Source-Count     : number of output files created
```

**Response:** `application/zip` binary download

## 🛠️ Development Guide

### Adding a New Tool

**Step 1 — Backend handler** `backend/tools/new_tool.py`:
```python
import pikepdf, tempfile, os

def process_new_tool(file_path: str, **kwargs) -> dict:
    """
    Always use pikepdf with recompress_flate=False for lossless output.
    NEVER use Ghostscript with -sDEVICE=pdfwrite (re-encodes streams).
    """
    with pikepdf.open(file_path) as pdf:
        result = pikepdf.Pdf.new()
        # ... your logic here ...
        out_path = tempfile.mktemp(suffix='.pdf')
        result.save(out_path, recompress_flate=False)
    return {"output": out_path, "pages": len(result.pages)}
```

**Step 2 — Flask route** in `backend/app.py`:
```python
@app.route('/api/new-tool', methods=['POST'])
def api_new_tool():
    f = request.files.get('file')
    if not f:
        return jsonify({"error": "No file"}), 400
    tmp = tempfile.mktemp(suffix='.pdf')
    f.save(tmp)
    try:
        result = process_new_tool(tmp, **request.form)
        return send_file(result['output'], as_attachment=True,
                         download_name='output.pdf',
                         mimetype='application/pdf')
    finally:
        if os.path.exists(tmp): os.unlink(tmp)
```

**Step 3 — Frontend folder** `frontend/tools/new-tool/`:
```html
<!-- index.html — CRITICAL: base href must be first after charset -->
<meta charset="UTF-8">
<base href="/tools/new-tool/">
<!-- ... rest of head ... -->
```

**Step 4 — TOOLS array** in `frontend/static/js/main.js`:
```javascript
{ name:'New Tool', category:'organize', path:'new-tool',
  desc:'Short description for search and card', icon:'fa-star',
  apiEndpoint:'/api/new-tool' }
```

### Critical Architecture Rules

```
RULE 1: LOSSLESS QUALITY
  ✅ Use: pikepdf.save(path, recompress_flate=False)
  ❌ Never: Ghostscript -sDEVICE=pdfwrite (re-encodes)
  ❌ Never: pikepdf without recompress_flate=False

RULE 2: CSS ANIMATIONS
  ✅ Use: CSS @keyframes with transform:translateY() only
  ❌ Never: opacity:0 in reveal/IO patterns (content hidden on slow loads)
  ❌ Never: gsap.from({opacity:0}) on above-fold elements

RULE 3: SCRIPT LOADING
  ✅ sounds.js → regular <script> (MUST initialize before user interactions)
  ✅ All external JS (GSAP, confetti, etc.) → defer
  ✅ Tool script.js → defer

RULE 4: TOOL PAGE SETUP
  ✅ <base href="/tools/<name>/"> must be FIRST element after charset meta
  ✅ [hidden]{display:none!important} in every tool's style.css
  ✅ All DOM refs initialized inside DOMContentLoaded (let D = null at top)

RULE 5: FONT AWESOME ICONS
  ❌ Never: background-clip:text on FA icons (breaks glyph rendering)
  ✅ Use: color: property or filter:drop-shadow()

RULE 6: PDFMINER IMPORTS
  ✅ from pdfminer.layout import LTTextBox
  ❌ Never: LTAnon, LTTable (don't exist in current version)
```

### Quality Score Algorithm
```python
def estimate_split_quality_score(result: dict, in_bytes: int) -> tuple[int, str]:
    score = 100
    if result.get('method') != 'pikepdf':      score -= 15  # non-lossless method
    if result.get('pages_reencoded', 0) > 0:   score -= 20  # stream re-encoding
    if result.get('compression_applied'):       score -= 10  # unintended compression
    # ...
    grade = 'A+' if score >= 95 else 'A' if score >= 85 else 'B' if score >= 70 else 'C'
    return score, grade
```
