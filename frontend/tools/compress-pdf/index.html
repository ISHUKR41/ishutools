The 12-Engine Pipeline

Detecting engines…
#1
pikepdf
Pure lossless stream recompression. DEFLATE-9 + ObjStm. Zero image re-encoding. Used for ALL presets.
All presets
#2
Ghostscript
Industry-standard PDF distiller. /screen /ebook /printer presets for aggressive image resampling.
Low/Med/Screen
#3
PyMuPDF (fitz)
Per-image DPI resampling preserving text and vectors. Bicubic interpolation for best visual quality.
Low/Med/Screen
#4
qpdf
Stream recompression + linearization. Safe for all presets — no image resampling.
All presets
#5
mutool
MuPDF garbage collection + stream compression. Fast, reliable, safe for all presets.
All presets
#6
Pillow
Progressive JPEG + optimal subsampling per image. Best for photo-heavy PDFs.
Low/Med/Screen
#7
pypdf
compress_content_streams() + orphan object removal. Safe for all presets.
All presets
#8
Deduplication
MD5 hash-based identical image removal. Completely lossless — zero quality impact.
All presets
#9
Content Streams
DEFLATE-9 recompression of PDF drawing commands. Safe for all presets.
All presets
#10
GS + pikepdf Chain
Double-pass: Ghostscript distill → pikepdf stream recompress. Often best result for screen/low.
Low/Med/Screen
#11
GS + fitz Chain
Ghostscript distill then PyMuPDF image polish — aggressive double-pass for screen mode.
Screen only
#12
Pillow + pikepdf
Pillow JPEG optimisation followed by pikepdf stream recompression. Best for photo PDFs.
Low/Med/Screen
