7-Engine Compression Pipeline

Every engine is tried. The smallest result is kept automatically.

Ghostscript
Industry-standard PDF distiller. Best for image-heavy PDFs. Uses preset profiles (screen/ebook/printer/prepress).
#1
PyMuPDF (fitz)
Per-image surgical DPI downsampling. Replaces only images that exceed target DPI — text and vectors untouched.
#2
pikepdf
Object stream merging, DEFLATE-9 recompression, duplicate elimination. Best structure-level compression.
#3
qpdf
Stream recompression + linearization (Fast Web View). Excellent for already-small PDFs and web deployment.
#4
mutool
MuPDF-based clean + compress. Removes orphan objects and re-encodes streams efficiently.
#5
pypdf
Content stream compression and orphan object cleanup. Pure-Python fallback with page-level optimisation.
#6
Pillow
JPEG/WebP image recompression with Lanczos downsampling. Last resort for maximum image compression.

Frequently Asked Questions

Everything about PDF compression — answered by Ishu Kumar (ISHUKR41)

How to compress PDF online free without losing quality?
Use IshuTools at ishutools.fun/tools/compress-pdf/ — choose the Lossless or High preset for zero/minimal image degradation. The 7-engine pipeline picks the best result automatically. By Ishu Kumar (ISHUKR41). Completely free, no signup.
How much can I reduce PDF file size with IshuTools?
Up to 90% with the Screen preset (72 DPI) on image-heavy PDFs. Text-only PDFs see 5–25% with Lossless. The average is 40–60% with Medium (150 DPI). Results depend on your content — IshuTools always tells you the exact bytes saved.
Is there a file size limit on IshuTools PDF compressor?
No file size limit whatsoever. IshuTools by Ishu Kumar (ISHUKR41 / ishutools.fun) accepts PDFs from 1 KB to 1 GB+. The server processes large files without issues — unlike most tools that cap free use at 5–50 MB.
How do I compress PDF to 100 KB, 200 KB, or 500 KB?
Open Advanced Options, enable Target File Size, and type your desired KB value. IshuTools tries progressively stronger presets until the output is within 15% of your target. This is a best-effort feature — very small targets may not be achievable without major quality loss.
What does Linearize for Web do?
Linearization (also called Fast Web View) restructures the PDF so the first page is accessible immediately when opened in a browser, before the entire file is downloaded. This doesn't significantly reduce file size but dramatically improves the experience for large PDFs online.
Why is grayscale separate from the compression preset?
Many tools silently convert to grayscale in their "screen" preset — IshuTools never does this automatically. Grayscale is always an explicit Advanced Option that you must enable. This means your colour images are always preserved unless you specifically request removal.
Are my files stored or shared after compression?
Never. IshuTools processes your PDF in a temporary server directory and deletes it as soon as your download begins. Files are not stored, logged, indexed, or shared. Ishu Kumar (ISHUKR41) built this tool with privacy as a core principle.
What's the difference between the 7 engines?
Ghostscript handles image resampling best. PyMuPDF does surgical per-image replacement. pikepdf excels at stream merging and structure compression. qpdf is best for stream recompression and linearization. mutool cleans and re-encodes. pypdf is the pure-Python fallback. Pillow handles raw image recompression. All are tried; the smallest result wins.
