PDF Compression Technology — Detailed Explanation

Ghostscript (/screen, /ebook, /printer, /prepress): Industry-standard PostScript interpreter that re-renders and re-encodes the entire PDF. The /screen preset produces the smallest output (72 DPI images) while /prepress produces print-quality output with minimal compression.

PyMuPDF (fitz) image recompression: Extracts each embedded image individually, applies LANCZOS downsampling to the target DPI, re-encodes with progressive JPEG at the specified quality level, and embeds only if the new version is smaller. This is the most granular image compression approach.

pikepdf object stream optimization: Uses the QPDF library to merge PDF indirect objects into compressed object streams, apply DEFLATE level-9 compression to all content and font streams, and remove redundant cross-reference tables. This is particularly effective for text-heavy documents.

qpdf linearization: Rearranges the PDF byte layout for optimal web delivery — page 1 bytes come first, enabling browsers to render the first page before the entire file is downloaded.
