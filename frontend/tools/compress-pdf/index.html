7-ENGINE PIPELINE

Enterprise-Grade Compression Technology

Every engine runs on your file. The smallest valid result is kept automatically.

👻
Ghostscript
Industry-standard PDF distiller. 5 quality profiles (/screen to /prepress). Best for image-heavy PDFs.
#1
🖼
PyMuPDF (fitz)
Per-image DPI downsampling with Lanczos resampling. JPEG re-encode at target quality. Handles page-level image replacement.
#2
🐟
pikepdf
Object stream merging, DEFLATE-9 recompression, duplicate stream removal. Best for structure-level + lossless gains.
#3
⚡
qpdf
Stream recompression, object stream generation, web linearization. Used for post-processing chain on best result.
#4
🔩
mutool
MuPDF-based clean pass — deduplicates, deflates, removes orphan objects. Very fast structure optimizer.
#5
📝
pypdf
Pure-Python PDF library. Compresses page content streams, removes orphan objects. Reliable pure-Python fallback.
#6
🎨
Pillow
Image processing for JPEG/WebP recompression. Handles grayscale conversion, DPI-aware resize with Lanczos filter.
